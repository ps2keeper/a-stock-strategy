# A股选股策略分析系统

一个面向 A 股交易实践的选股与持仓风险管理工具，基于 **多因子确认**、**三确认法则** 和 **纪律化风控** 设计。

项目包含：

- **选股分析**：K线形态、量价关系、均线系统、MACD / KDJ / RSI / 布林线
- **持仓管理**：持仓录入、现金记录、加权平均成本、最新价格估值
- **风险评估**：仓位比例、现金安全垫、单股集中度、止损触发预警、铁律合规检查
- **市场情绪**：上证 / 深证、涨跌停数、上涨下跌家数、活跃度

## 核心理念

> 宁可错过，不可做错；趋势为王，风控至上。

系统默认遵循以下交易纪律：

- 第一次入场最多 **3 成仓**
- 个股跌破入场价 **-7%** 默认止损
- 保留足够现金应对波动和黑天鹅
- 只在 **趋势确认 + 风险可控** 时考虑入场

## 功能概览

### 1. 多因子选股引擎

系统会对单只股票做综合评分，并给出：

- **综合评分**（0-100）
- **三确认法则** 是否通过
- **投资建议**：强烈买入 / 谨慎买入 / 持续观察 / 回避观望
- **建议止损位**

支持的策略因子：

- **K线形态**：锤头线、看涨吞没、早晨之星
- **量价关系**：放量上涨、连续价升量增
- **均线系统**：5/10/20/60 日线趋势确认
- **辅助指标**：MACD、KDJ、RSI(14)、布林线

### 2. 持仓与现金资产管理

支持本地持久化保存：

- 现金余额
- 股票代码 / 名称
- 持仓数量
- 成本价
- 最新价格
- 市值、浮盈亏、持仓占比

持仓数据使用浏览器 `localStorage` 保存，刷新页面不会丢失。

### 3. 风险评估面板

根据当前持仓与现金结构，自动评估：

- **总资产 / 股票市值 / 现金余额**
- **仓位比例** 与 **现金比例**
- **最大单股占比**
- **整体浮盈亏**
- **止损触发数量**
- **风险评分与风险等级**

风险等级分为：

- 低风险
- 中等风险
- 高风险
- 极高风险

并输出风控预警，例如：

- 仓位过高
- 单股过度集中
- 持仓触及止损线
- 总体浮亏超过警戒线
- 持仓数量过多难以跟踪

## 启动方式

### 1. 启动后端

```bash
cd /path/to/Accio
./backend-env/bin/python backend/app.py
```

默认地址：

- API: `http://localhost:5001`

### 2. 启动前端

```bash
cd /path/to/Accio/stock-strategy
npm install
npm run dev
```

默认地址：

- Web: `http://localhost:5173`

## 主要接口

### 获取单股分析

```http
GET /api/stock/kline?code=000001
```

### 获取市场概览

```http
GET /api/market/overview
```

### 批量获取最新价格

```http
POST /api/stock/prices
```

请求体：

```json
{
  "codes": ["000001", "600519"]
}
```

### 批量选股

```http
POST /api/screen/batch
```

## 使用说明

1. 在 **选股分析** 页输入股票代码，查看因子评分与图表
2. 在 **持仓管理** 页录入你的持仓与现金
3. 在 **风险评估** 页查看当前仓位、集中度、止损触发情况与整体风险等级
4. 根据系统建议调整仓位与风控策略

## 注意事项

- 行情数据来自 **AKShare**，不保证实时性
- 本系统是 **辅助决策工具**，不是收益承诺工具
- 所有建议都应结合个人交易体系与风险承受能力使用
- 使用前请确认你理解并接受止损纪律
